A sleek, modern event registration application built for workshops and challenges. Users can submit their details, choose an event, and have the data stored securely.
The app features a glass-style interface, animated background, and responsive design.

Live App: https://registration-beryl-seven.vercel.app/

🛠 Features
Modern glass UI with animated floating blobs
Fully responsive for mobile and desktop
Form validation for name, email, KU ID, and event selection
Supabase backend for secure and scalable database storage
Smooth UI feedback on form submission
Hosted on Vercel for instant deployment
💻 Tech Stack
Layer	Technology
Frontend	HTML5, CSS3, JavaScript ES6
Backend / DB	Supabase (PostgreSQL)
Hosting	Vercel
Fonts	Google Fonts – Outfit
Optional Tools	Git, Node.js (for local dev)

Backend Language: No traditional server code is needed—Supabase handles the backend using PostgreSQL and Supabase Functions.
JavaScript in the frontend communicates directly with Supabase.
